The Velkoz Data Ingestion Project

A REST API service providing coellated web-extracted data.

The Velkozz data project is designed to extract, transform and aggregate data from across the web. The project is an open source project built using the Django-Rest-Framework (DRF). The Velkozz Data Project is made up of multiple servers and applications that: extract data via web scrapers, aggregate/transform and load data into backend databases. The DRF application serves as the front-end user interface into the Velkozz Data Project. The Documentation and structure of the project can be found here.
